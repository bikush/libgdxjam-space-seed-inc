# Space Seed Inc.

A hot-seat, 2 player asteroids game. 
Created with [libGDX](http://libgdx.badlogicgames.com) for [libGDXjam 2015/2016](http://itch.io/jam/libgdxjam).

## Setup

Created with the libGDX project creator. Builds with Gradle. Extensions used: Box2d, Freetype.

## Description

Game description can be found on [itch.io game page](http://bicoosh.itch.io/space-seed-inc)

Game entry to the jam is [here](http://itch.io/jam/libgdxjam/rate/51175). Voting until Feb 1st 2016.
